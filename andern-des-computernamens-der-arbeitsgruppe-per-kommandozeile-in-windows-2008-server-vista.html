Ändern des Computernamens per Kommandozeile in Windows

Neues Jahr, neue Aufgaben. Ich finde mich in jüngster Zeit öfters in der Konsole wieder. So komm'ts auch das ich über nette kleine Kommando's für ein wenig Server-Installation gestolpert bin. Zum ändern des Computernamens gibt es z.B. ein nettes Tool namens wmic. Damit kann man wunderbare Dinge anstellen (gegeben man hat die Rechte), denn es ist wie eine "Shell" für WMI.

Damit lassen sich unter Anderem der Computername...

C:\> wmic computersystem where name="%computername%" call rename name="meincomputername"

...und die dazugehörige Arbeitsgruppe oder Domäne...

C:\> wmic computersystem where name="%computername%" call joindomainorworkgroup name="meinearbeitsgruppe"

...relativ einfach und schnell ändern.
