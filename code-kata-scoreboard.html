Code Kata Scoreboard

This week we had a Coding Dojo in Munich and let me just start by saying that it was real funny and challenging at the same time. That by itself is a nearly regular thing for Coding Dojo's. Yet, this time we had some extra fun and extra challenge thanks to a special Code Kata developed for the Dojo. The kata I'm talking about is Scoreboard.

Before I continue on describing the kata, let me give credits to the original inventor of the kata. Just the very same as it was with the incredible PickAkin kata, this kata was not my idea but the idea of smart people around me. The kata was invented specifically for this Coding Dojo by Dennis Wagner, who is a regular Dojo participant in Munich for over a year. Since the UEFA Euro 2012 Football Championship is being played this summer, he thought if it was possible to integrate this topical theme into our Dojo activities. The very successful result of this attempt is in fact the Scoreboard kata.

At our Coding Dojo, we had three teams tackling the kata and to all our surprise the kata turned to be a very good source for interesting questions and approaches within the field of OO Design, type layout and test structuring. I wholeheartedly can recommend this Code Kata to anyone. Enjoy!

Scoreboard

You are an aspiring mobile developer who just got a little product description from a very big organization to develop an App for. The name of the organization is no less than the great UEFA himself, asking you to develop an App which is able to display Scoreboards of groups for the Euro 2012 Championship. As an added feature, the App should be able to display Scoreboards for not only the actual match results, but any "custom" match results the user may enter.

As one of the first and most important things, you need to develop the scoring rules. It's crucial that the scoreboards actually display positions of teams without any error. Therefore, the official scoring system of the UEFA needs to be implemented. The scoring rules are described in detail in the official Regulations of the Euro 2012 Football Championship, Section 8.

As a nice to have suggestion, the sponsor of the UEFA would be very pleased if your implementation of the scoreboard rule system would be flexible and interchangable, so that regularatory changes or even different rules from other tournaments such as the FIFA World Championship could be applied.
