.NET Coding Dojo

Die .NET Coding Dojo Gruppe aus München ist das erste .NET Coding Dojo Deutschlands. .NET Entwickler und .NET Interessierte aller Erfahrungsbereiche treffen sich beim .NET Coding Dojo monatlich um gemeinsam die Kenntnisse über Software-Entwicklung und das .NET Framework zu erweitern. Das Ziel des Treffens ist es, in einer gelösten, lockeren Atmosphäre gemeinsam ein Code Kata (eine Programmieraufgabe) zu stemmen und dadurch Neues zu erlernen.

Das .NET Coding Dojo ist eine kostenlose und unverbindliche Veranstaltung. Von der Community, für die Community. Auf diesen Seiten sind nähere Informationen zum .NET Coding Dojo zusammengestellt. Darüber hinaus organisiert sich das .NET Coding Dojo online mit Hilfe von Xing & Facebook.
