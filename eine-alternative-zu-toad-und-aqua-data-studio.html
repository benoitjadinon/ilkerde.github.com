Eine alternative zu TOAD und Aqua Data Studio…

ist sicherlich der von Oracle frei verfügbare SQL Developer. Er ist zwar nicht so mächtig wie TOAD, hat auch leider nicht so angenehme Analyzer-Funktionen wie Aqua Data Studio, ist aber dennoch ein kleines feines Tool, um schnell an Informationen auf einer Oracle-Datenbank heranzukommen.

Wie nahezu alles von Oracle ist auch der SQL Develoer in Java geschrieben. Die Geschwindigkeit ist akzeptabel, das UI ist relativ einfach zu bedienen. Wer also mal ab und zu auf einer Oracle-DB arbeitet, der ist mit diesem Werzeug meiner Meinung nach recht gut bedient.

written on 14 June 2007
by Ilker Cetinkaya.
