The Documentation

I once just thought to myself: "Why on earth should I need a documentation?". Well, today it turns out that I think to myself: "Wow, how does this thing work again? And why?".

This is not going to be a very thorough or complete documentation, though. All I want to place in here is just what I've written so far (for example in the master in-place documentation) or just what I think is needed to be written down for reference purposes. Yeah, that's not much of big news. Well, almost.

The "big news" actually is that I finally decided to dedicate a whole section of my website for its own documentation. That being said, I'm somewhat shifting away from in-place documentation to a dedicated documentation within a designated docs folder.
