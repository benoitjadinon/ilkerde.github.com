The Inconspicuous Code Files – ICF

Es tut sich was innerhalb von Works. Letztens habe ich bei einem kleinen Developer-Frühstücks-Talk das Thema "Utils" angerissen. Es scheint ja wohl so zu sein, dass wirklich jeder seinen eigenen kleinen Utils-Namespace hat.

Bei mir ist das ja auch nicht anders. Wenn man mal ein paar Projekte hinter sich hat, kommen ja auch ein paar Kleinigkeiten zusammen. Naheliegend also die Idee, die zusammengehäuften Code-Fragmente mal in Works einzustellen.

Gesagt getan. Ab heute gibt es das "Inconspicuous Code Files"-Projekt in Works. Ja, es gibt einfachere Bezeichnungen für Util-Klassen, also gleich hinterher die Abkürzung: ICF. Ist ja auch ein wenig flüssiger in der Aussprache.

Zum Start der ICF habe ich mir einen - meiner Meinung nach - "Klassiker" ausgesucht: EventArgs. Vor allem bei kleineren und mittelgroßen Projekten ist es eine hilfreiche Klasse, die einem unnötiges Getippe erspart. Und für die Konventionsfanatiker, die trotzdem eindeutige, klare EventArgs haben wollen, ist es immerhin eine gute Basis-Klasse.

Mit der Zeit werde ich sicher auch den Rest peu à peu einstellen - aber es gibt ja auch so noch einiges zu tun.

written on 07 March 2007
