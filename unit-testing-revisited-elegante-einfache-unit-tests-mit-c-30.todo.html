Unit Testing Revisited – Elegante & einfache Unit Tests mit C# 3.0

Nun, Unit Tests sind ja mittlerweile in der professionellen Software-Entwicklung Standard - schon fast ein alter Hut. Gleiches gilt natürlich für Unit Tests in der .NET-Welt, denn Tools wie NUnit sind ja seit langer Zeit schon Teil des Standard-Repertoires eines .NET-Entwicklers.

Aber es gibt Neuigkeiten, denn in jüngster Zeit ist wieder ein wenig Bewegung in das Thema Unit Testing mit .NET gekommen. Das liegt vor Allem an der Veröffentlichung von C# 3.0 / .NET Framework 3.5. Neue Tools wie MoQ, Autofac oder XUnit.net bringen frischen Wind in die Unit-Test-Bude.

Ich habe mich, auch auf Grund der Aktualität in meinem Umfeld, mal näher mit o.g. neuen Tools beschäftigt. Meine daraus gewonnenen Erkenntnisse und Erfahrungen habe ich im Works-Bereich festgehalten:

Unit Testing Revisited - Die Unit Test Evolution mit C-Sharp 3.0 ist ein Artikel über das Unit Testing unter Einsatz von MoQ und Autofac. Für die Code Samples musste natürlich wieder das berühmt-berüchtigte "Calculator"-Beispiel herhalten.
